# Airペイ手数料率一覧

## キャンペーン料率（2.48%）

対象ブランド:
- Visa
- Mastercard(R)
- JCB
- American Express
- Diners Club
- Discover

参照: https://airregi.jp/payment/discountprogram/

## 通常料率

### iD / QUICPay（3.24%・非課税）
- iD
- QUICPay

### 交通系電子マネー（2.95% + 税 = 3.245%・課税）
- Suica, PASMO, Kitaca, TOICA, manaca, ICOCA, SUGOCA, nimoca, はやかけん

### QR決済（2.95% + 税 = 3.245%・課税）
- PayPay
- d払い
- 楽天ペイ
- au PAY
- COIN+
- WeChat Pay
- Alipay+

参照: https://faq.airpayment.jp/hc/ja/articles/218985588

## 計算ルール

1. 入金サイクルごとに決済ブランド別の売上を合計
2. 合計金額に手数料率を乗算
3. 小数点以下は切り捨て

## 振込手数料

振込手数料は無料（どの銀行口座でも）
※ゆうちょ銀行は振込口座に指定不可
