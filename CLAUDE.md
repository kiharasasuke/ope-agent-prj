# 経理データ処理プロジェクト

## プロジェクト概要

このプロジェクトは、銀行取引明細（PDF）を解析し、会計処理に使用できるCSVデータを生成する経理業務支援システムです。

## フォルダ構成

```
ope-agent-prj/
├── 01_預金/          # 銀行取引明細PDF（入力データ）
├── agent_docs/       # 処理ルール・プロンプト定義
├── output/           # 処理結果の出力先
│   └── 01_預金/      # 預金データの出力
└── CLAUDE.md         # このファイル
```

## 処理対象データ

- `01_預金/`: 各店舗の銀行取引明細PDF
  - 呑家（2025年11月・12月）
  - 西口アオギリ（2025年12月）
  - ホドケバ（2025年11月・12月）

## 処理ルール

詳細な処理ルールは `agent_docs/g_銀行預金総合データ処理.md` を参照。

主な処理フロー:
1. PDFからOCR読み取り
2. レイアウト判別・正規化
3. 残高整合性チェック
4. CSV出力（日付, 摘要, 出金額, 入金額, 残高）

## 出力形式

```csv
日付,摘要,出金額,入金額,残高
```

- UTF-8エンコーディング
- 数値は整数のみ
- 空欄は0
