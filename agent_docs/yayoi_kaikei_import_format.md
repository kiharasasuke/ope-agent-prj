# 弥生会計 仕訳インポート形式 (Yayoi Kaikei Import Format)
弥生会計（特に弥生会計 オンラインや最近のデスクトップ版）で一般的に使用される仕訳データのインポート形式（CSV/テキスト）です。
## ファイル形式の要件
- **形式**: CSV (カンマ区切り) または テキスト (タブ区切りなど、通常はCSV推奨)
- **文字コード**: Shift-JIS (CP932)
  - UTF-8（BOM付）では弥生会計にインポートできない
  - Shift-JISで出力すると㈱などの特殊文字が文字化けする可能性があるため注意
- **改行コード**: CRLF
  - CRのみだとインポートできない
  - Excelから手動で保存したCSVファイルはCRLFになる
- **ヘッダー**: 通常は不要（データ行のみ）、ただしソフトによってはヘッダー行を無視する設定も可能。
- **データの囲み**: 項目内にカンマが含まれる場合はダブルクォーテーション `"` で囲む。
## 項目一覧 (全25項目)
| 列 | 項目名 | 必須 | 説明 |
| :--- | :--- | :--- | :--- |
| A | 識別フラグ | 〇 | 仕訳の種類を識別するコード (例: 2000, 2110, 2111 など) |

| C | 決算 | | 決算仕訳の場合に「本決」などを入力。通常は空欄。 |
| D | 取引日付 | 〇 | YYYY/MM/DD 形式など。 |
| E | 借方勘定科目 | 〇 | 借方の勘定科目名。 |
| F | 借方補助科目 | | 借方の補助科目名。 |
| G | 借方部門 | | 借方の部門名。 |
| H | 借方税区分 | 〇 | 課税売上、課税仕入などの税区分コードまたは名称。 |
| I | 借方金額 | 〇 | 借方の金額。 |
| J | 借方税金額 | | 借方の消費税額。 |
| K | 貸方勘定科目 | 〇 | 貸方の勘定科目名。 |
| L | 貸方補助科目 | | 貸方の補助科目名。 |

## 識別フラグの例
- `2000`: 1行のみの単一仕訳 (振替伝票以外)
- `2110`: 複合仕訳 (振替伝票) の1行目
- `2111`: 複合仕訳 (振替伝票) の2行目以降
- `2101`: 複合仕訳 (振替伝票) の最終行
## 注意点
- **金額**: カンマなしの数値で入力します (例: `10000`)。
- **空欄**: 該当しない項目は空欄 (カンマのみ `,,`) にします。
- **バージョン差異**: 弥生会計のバージョン（05, 07, 12, 24など）や製品（スタンダード, プロフェッショナル, オンライン）によって、項目数や順序が微妙に異なる場合があります。上記は近年の一般的な形式（特にオンライン版やインボイス対応版）に基づいています。古いバージョンでは項目数が少ない場合があります。
